#最简易的音频视频(AV)播放器


-------------------------------------------


**HTML5(手机)下的回调事件callbacks:**


*  canplay    当音/视频能播时
*  timeupdate 当音/视频开始或正在播时
*  pause      当音/视频暂停时
*  ended      当音/视频结束时
*  error      当音/视频出错时

**flash(PC)**

* playerStart 当音/视频开始时，参数总时长
* playerProcess  当音/视频正在播时，第一个参数为当前播放位置，第二个参数为当前播放的百分比
* playerEnd 当音/视频结束时

*  flashInsertComplete 全局方法，当flash播放器插入完成后
